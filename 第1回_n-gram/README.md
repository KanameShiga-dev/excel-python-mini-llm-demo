- `excel_n_gram_demo.xlsx`: デモ用Excel。`00_README`、`01_Demo`、`13_SERIES_ROADMAP` などを含む。

- `excel_n_gram_demo_design.md`: 第1回デモの設計説明。
- `series_roadmap.md`: 全10回のシリーズロードマップ。
- `thumbnail.jpg`: サムネイル画像。
